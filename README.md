## 目录部署

| 目录名称            | 项目名称     | 开发周期 | 启动指令   |
| ------------------- | ------------ | -------- | ---------- |
| wuwandianyingapp    | 五万电影app  | 5        | yarn serve |
| wuwandianyingpc     | 五万电影pc   | 5        | yarn serve |

## 启动项目

如何启动: yarn serve

如何测试: yarn run

如何发布：yarn bulid
